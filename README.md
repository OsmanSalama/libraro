# Libraro

Libraro is a user-friendly Android app designed for book lovers to easily browse, read, and manage their digital library. With a focus on accessibility and personalized reading experiences, Libraro offers a range of features to enhance your literary journey.

## Features

- **Browse Digital Library**: Easily browse and discover books in your collection
- **Read Books**: Seamless reading experience with customizable display options
- **Manage Collection**: Organize and manage your digital library
- **Accessibility**: User-friendly interface designed for accessibility
- **Personalized Experience**: Tailored features to enhance your reading journey

## Tech Stack

- **Language**: Kotlin
- **Platform**: Android

## Getting Started

### Prerequisites

- Android Studio
- Java Development Kit (JDK)
- Android SDK

### Installation

1. Clone the repository:
```bash
git clone https://github.com/OsmanSalama/libraro.git
cd libraro
```

2. Open the project in Android Studio

3. Sync Gradle dependencies

4. Build and run the app on an emulator or physical device
